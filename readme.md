# 🚀 JavaScript Module 08 - Fundamental Concepts of Conditionals

This repository contains my practice solutions for **Module 08: Fundamental Concepts of Conditionals** from the **Programming Hero AI Driven Full Stack Web Engineering** course.

The goal of this module was to understand how JavaScript makes decisions using comparison operators, logical operators, conditional statements, and the ternary operator.
